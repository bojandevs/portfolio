About

I am a Front-End Developer dedicated to creating clean, responsive web interfaces using HTML, CSS, and JavaScript. Over the past two years, I have developed a portfolio of projects ranging from task managers to API-driven applications, each reinforcing best practices in code structure and user experience.

My interest in web development began when I realized the power of building interactive applications from the ground up. Since then, I have focused on:

  • Implementing semantic markup and scalable CSS layouts with Flexbox and Grid.
  • Mastering vanilla JavaScript: DOM manipulation, asynchronous requests, and ES6+ syntax.
  • Exploring React and Next.js to understand component-based architecture and server-side rendering.
  • Maintaining clear, concise documentation and commit history in GitHub repositories.
  • Ensuring accessibility standards and mobile-first design principles in every project.

When I am not coding, I experiment with advanced CSS methods, stay informed about the latest JavaScript features, and contribute to open-source projects whenever possible. If you would like to review my work or discuss a potential collaboration, please visit the Projects page or get in touch.
