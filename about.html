About

Greetings! I’m a Front-End Developer who loves crafting neat, responsive web interfaces using HTML, CSS, and JavaScript. Over the past couple of years, I’ve built a small but growing portfolio of personal projects—ranging from simple to-do lists to interactive APIs—each teaching me something new about clean code and user experience.

My journey into web development started when I realized how empowering it is to build something from scratch and see it live on the internet. Since then, I’ve focused on:

  • Writing semantic markup and organized CSS (Flexbox & Grid).
  • Mastering vanilla JavaScript fundamentals: DOM manipulation, fetch/async, ES6+ syntax.
  • Diving into React and Next.js basics to understand component-based architecture.
  • Keeping my GitHub repos polished with clear commit messages and robust README files.
  • Ensuring accessibility and mobile-first design in every new project.

Outside of coding, I enjoy experimenting with new CSS techniques, reading about the latest JavaScript features, and contributing to open‐source whenever I can. If you’re curious about my work or want to team up on a project—feel free to check out my Projects or get in touch!
